The following project features a 2D platform game developed with the help of the sample project given in the book "SDL Game Development by Shaun Mitchell" . All the artworks are downloaded under CC0 license from "opengameart.org".

The Goal of the project is to reach the end of the level avoiding various enemies keeping your health. Beware of the animals that are keen to attack. The bar on the top left corner indicates your health. 

The project is developed in a purely Object Oriented way which makes use of the following technologies.
1. C++ Programming
2. SDL(Simple DirectMedia Layer 2.0)
3. TinyXml (Xml parser for C++)
4. SDL Mixer and SDL Image
5. Tiled Tilemap editor
